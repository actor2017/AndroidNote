static class ViewHolder {//静态
    private TextView tv;
    public ViewHolder(View convertView) {
        tv = (TextView) convertView.findViewById(R.id.tv_address);
    }
}

内存只加载一次,静态内部类不需要持有外部对象的引用,防止内存泄露.
findViewById的次数是屏幕条目数+1条(convertView == null),然后复用缓存(convertView != null)
